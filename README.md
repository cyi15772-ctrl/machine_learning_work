# 机器学习与神经网络导论期末作业
文件介绍
datasets文件夹下面为输入数据集，其中enron1文件夹下的全部数据为第六题的输入数据，附件1文件夹下的回归预测.xlsx文件为第九题的输入数据，说明.docx文件为该数据的描述文件。
第六题_spam_classify_change.ipynb文件为第六题的实现代码，记得替换其中的输入数据文件夹路径为你保存数据文件的路径
第九题_reg_predict.ipynb文件为第九题的实现代码，记得替换其中的输入数据文件夹路径为你保存数据文件的路径
test_sample_errors.csv文件是第九题代码的输出文件，它存储了每个测试样本的真实值、预测值和误差
